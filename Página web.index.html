Conceptos Básicos

La coctelería es el arte de mezclar bebidas y crear combinaciones únicas de sabores, aromas y presentaciones.

  • Cóctel clásico: receta tradicional con ingredientes definidos.
  • Mocktail: cóctel sin alcohol.
  • Decoración: uso de frutas, hierbas y elementos visuales para embellecer la bebida.

Técnicas de Coctelería

Algunas técnicas básicas que todo bartender debe conocer:

  • Shaker: agitar los ingredientes en coctelera para mezclarlos y enfriarlos.
  • Muddling: machacar frutas o hierbas para liberar sabores.
  • Layering: formar capas con líquidos de distinta densidad.
  • Straining: colar para retirar hielo o residuos sólidos.
